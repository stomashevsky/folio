# SEO and AI Visibility Monitoring Guide

This document outlines the monitoring strategy for tracking Folio's search engine and AI platform visibility.

## Google Search Console Setup

### Initial Setup
1. Go to [Google Search Console](https://search.google.com/search-console/)
2. Add property for `https://folio.id/`
3. Verify ownership using DNS TXT record or HTML file
4. Submit sitemap: `https://folio.id/sitemap.xml`

### Key Metrics to Monitor
- **Indexing Status**: Ensure all pages are indexed (target: 70+ pages)
- **Coverage**: Check for crawl errors, redirect issues
- **Performance**: Monitor clicks, impressions, CTR, and average position
- **Core Web Vitals**: LCP, FID, CLS scores
- **Mobile Usability**: Ensure no mobile-specific issues

### Weekly Checks
- [ ] Review indexing status for new pages
- [ ] Check for any crawl errors
- [ ] Monitor keyword rankings for target terms
- [ ] Review mobile usability report
